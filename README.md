# Car Rental Invoice System

A pixel-perfect, A4 print-ready car rental invoice template built with HTML and CSS. Features a bilingual layout (Arabic & English) designed for professional automotive businesses.

## Features

- **A4 Print Optimized**: Meticulously designed for physical printing with correct margins and scaling.
- **Bilingual Interface**: Full support for English and Arabic text side-by-side.
- **Professional Layout**:
  - Structured data tables for Vehicle, Renter, and Sponsor info.
  - Interactive Car Diagram with checklist.
  - Detailed Terms & Conditions in Arabic.
  - Dedicated Operations section (Charges, Return, Departure).
- **Responsive Design**: Uses modern CSS Flexbox and Grid for consistent rendering.
- **Dynamic Assets**: External SVG car diagram for sharp, scalable visuals.

## Project Structure

- `index.html`: Main invoice structure and content.
- `styles.css`: Styling, layout logic, and print optimizations.
- `car-diagram.svg`: Externally linked car inspection diagram.

## Usage

1. Open `index.html` in any modern web browser.
2. To print, use `Ctrl+P` (Windows) or `Cmd+P` (Mac).
3. Ensure "Background Graphics" is enabled in print settings for the best visual experience.
